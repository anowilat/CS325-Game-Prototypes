As Seen On TV

  • High Concept:
    Side scrolling stealth game where players must control an anonymous individual hunted by
    unknown forces meaning to steal their memories.

  • Theme:
    • Television Is Broadcasting Actual Memories

  • Mandated Variety:
    • Input: Keyboard buttons
    • Randomness: Some Randomness
    • Play styles: --
    • Genres: Side-scroller, Stealth

  • Player Experience Goals:
    • Appeal to players through aesthetic and dark themes.
    • Provide players with an entertaining side scroller fused with stealth mechanics
    • Instill a sense of suspense to get hero to safety.

  • Gameplay:
    Players control a side scrolling character using keyboard arrow input.
    The goal of the game is to reach the distance required before a set amount of time; while
    avoiding thugs running in the opposite direction by hiding behind crates.
    Players can move up and down the alleyway using the up and down keys, and hide behind crates
    by pressing the left arrow key when close enough, then pressing the right arrow key to resume
    running. If players do not manage to hide before a thug gets close enough, the player is killed.

  • Prototype Goal:
    • To digitally implement a physical prototype and compare the evolution of the game mechanic.

  • Features:
    The gameplay mechanics of the game are simple, with the exception that the player is able to hide
    and temporarily stop the scrolling mechanic. The bulk of the work was spent presenting the game
    in an artistic way, and polishing was done in the way of presentation, sound, and aesthetic.

  • Setting/Premise:
    The setting for the game is in an urban area; potentially between streets and alley ways.

  • Story:
    An unknown individual is running from an organization which steals peoples memories to broadcast them
    on television. The memories owners are never heard from again. While the individual runs from the
    goons, it might be noticed that the situation is observed from a television; which brings to question
    the heros survival...

  • Target Audience:
    This game is may not be suitable for all ages, but players who enjoy side scrollers or stealth might
    enjoy the gameplay.

  • Play Time:
    The game should not take very long, the player can reach an end goal and win.

  • Strategies:
    Gaurds footsteps can be heard a couple seconds before they appear on screen; giving players time to
    properly react if a crate is nearby.
    Players must pay attention to the time remaining versus the distance still left to cover. Hiding frequently
    can greatly decrease the time left; especially when a gaurd is not actually incoming.
